This is an android application boilerplate follow clean architecture

Libraries:
1. Kotlin + Java8
2. Koin: A library for dependencies injection, a lightweight alternative solution for Dagger2
3. ButterKnife: UI injection
4. RxKotlin: A library support for reactive programming with Kotlin
5. Timber: A library logging for app.
6. Glide: Image loading library
7. Mockito: Library for Unittest
8. Firebase: Crash analytics
9. PMD: Code style check
10. Sonarquebe: Tool for check code quality
